# 프로젝트 시스템 개발 플랜

## 📋 목표
프로젝트 3가지 타입 구현:
1. **일반 폴더**: Inbox 태스크 그룹핑
2. **학생 시간표**: 자동 수업 생성 및 출결 관리
3. **루틴/습관**: 자동 루틴 생성 및 추적

---

## 🎯 Phase 1: 데이터 구조 확장 (기초)

### 1-1. Project 타입 확장
- [x] `type` 필드 추가: 'folder' | 'student' | 'habit'
- [ ] 학생 시간표 전용 필드 추가
- [ ] 루틴/습관 전용 필드 추가

### 1-2. Task 필드 확장

- [ ] 공통: `is_auto_generated` (자동 생성 여부)

**우선순위**: 🔴 HIGH
