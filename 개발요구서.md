# Final UI Layout & Design Specs

## 1. Design System (Vercel-like Aesthetic)
- **Theme:** Ultra-minimalist, clean, professional.
- **Components:** Shadcn/UI (Radix UI based).
- **Typography:** Inter or Geist Sans (Crisp, readable sans-serif).
- **Colors:** Monochromatic (White/Black/Grays). Use subtle borders (`border-gray-200`) to define spaces instead of heavy backgrounds or shadows.
- **Spacing:** Compact density to show maximum information without looking cluttered.

## 2. Layout Grid (3-Column Independent Panels)
Use `react-resizable-panels` to create a flexible 3-pane layout.

### **[Left Panel: Inbox & Focus] (Width: ~20%)**
*Container with Flex-col layout*
1.  **Top (Focus Zone):**
    - Title: "Today's Top 5"
    - Content: A static list showing only the 5 most important tasks.
2.  **Center (Collection Zone):**
    - Title: "Inbox"
    - Content: Scrollable list of unsorted tasks/texts.
    - Styling: Simple text list with minimal styling.
3.  **Bottom (Capture Zone - Sticky):**
    - UI: A clean textarea input field (like a chat input).
    - Action: Enter key submits text to 'Inbox'.

### **[Center Panel: The Stage] (Width: ~60%)**
*The Main Workspace*
- **Component:** Full-height Weekly Calendar.
- **Granularity:** 10-minute time slots (Vertical grid).
- **Interaction:**
  - Drag & drop tasks from Left Panel (Inbox) to specific time slots here.
  - Click empty slot to quick-add.
  - Drag blocks to resize/move.

### **[Right Panel: Context & Storage] (Width: ~20%)**
*Container with Flex-col layout*
1.  **Top (Navigation):**
    - Component: Mini Calendar Widget (Month view).
    - Action: Clicking a date jumps the Center Panel to that date.
2.  **Center (Projects):**
    - Component: Collapsible Accordion list of Projects.
    - Content: Drag tasks here to classify them.
3.  **Bottom (Memos):**
    - Component: Folder Tree structure for Memos/Notes.
    - Content: Tag-based or Folder-based storage for non-actionable items.

## 3. Responsive Behavior
- **Mobile:** Show ONLY [Left Panel] by default. Use a bottom navigation bar to switch to [Center Calendar].
- **Desktop:** Show all 3 panels expanded.
